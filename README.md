# FutureTweet

**Send tweets in the future, with CRON-like syntax**

## Getting Started

1. 
  Create a Twitter app and set the following env variables:
    ```
      TWITTER_CONSUMER_KEY
      TWITTER_CONSUMER_SECRET
      TWITTER_ACCESS_KEY
      TWITTER_ACCESS_SECRET
    ```

  2.
    Get dependencies
    
        mix deps.get



  3.
    Run server
    ```
      iex -S mix 
    ```
